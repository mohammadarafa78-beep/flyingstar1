# FlyingStar - حساب عهدة الشاحنة

موقع/تطبيق ويب (PWA) لحساب عهدة الشاحنة ومصاريفها، قابل للتثبيت على الموبايل ويشتغل أوفلاين.

## الملفات

- `index.html` — الصفحة الرئيسية (حاسبة العهدة والمصاريف)
- `reports.html` — صفحة التقارير (كل الرحلات المحفوظة مع فلترة بالتاريخ والسائق)
- `manifest.json` — إعدادات تثبيت التطبيق كـ PWA
- `sw.js` — Service Worker لتشغيل التطبيق أوفلاين وتسريع التحميل
- `icons/` — أيقونات التطبيق بمقاسات مختلفة

## الرفع على استضافة

الموقع static بالكامل (HTML/CSS/JS بدون سيرفر أو قاعدة بيانات)، فبيشتغل على أي استضافة ملفات ثابتة، زي:

- **GitHub Pages**: ارفع الملفات في مستودع، فعّل Pages من الإعدادات، واختر الفرع/المجلد.
- **Netlify / Vercel**: اسحب المجلد بالكامل (drag & drop) في لوحة التحكم.
- **أي استضافة عادية (cPanel وغيرها)**: ارفع كل الملفات والمجلدات زي ما هي داخل `public_html` (حافظ على مسار `icons/` كما هو).

مهم: حافظ على أسماء الملفات ومساراتها كما هي (`icons/...`) لأن `manifest.json` و`sw.js` بيشيروا لها بنفس المسار النسبي.

## تسجيل الدخول

التطبيق حاليًا من غير تسجيل دخول، لأن كل البيانات بتتخزن محليًا على متصفح/جهاز المستخدم فقط (مفيش سيرفر أو حسابات مستخدمين). لو حبيت مستقبلًا تضيف حماية بسيطة (كلمة سر واحدة لقفل الشاشة)، أو نظام حسابات حقيقي (يحتاج سيرفر وقاعدة بيانات)، ابعتلي وهساعدك تضيفها.

## تخزين البيانات

البيانات (الرحلات المحفوظة) بتتخزن في `localStorage` بتاع المتصفح. يعني:
- بتفضل محفوظة حتى لو قفلت المتصفح أو الموبايل.
- بتتمسح لو المستخدم مسح بيانات المتصفح، أو استخدم متصفح/جهاز تاني.
- مفيش مزامنة بين الأجهزة المختلفة (كل جهاز له بياناته الخاصة).

## الباك إند (اختياري)

مضاف مجلد `backend/` فيه باك إند PHP + MySQL عام (تسجيل مستخدمين، حفظ رحلات
في قاعدة بيانات حقيقية، إرسال إيميلات). لسه مش متوصل بالواجهة تلقائيًا —
تفاصيل التركيب والـAPI في `backend/README.md`.
